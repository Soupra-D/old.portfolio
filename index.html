Projet Machine Learning : Reconnaissance d'émotions

Lors de ma formation à l'ESIEA, j'ai eu un projet de groupe à réaliser en Python. Le but était de reconnaitre les émotions des gens, à partir de leur photo et d'un ensemble de points. Le tout grâce aux algorithmes de Machine Learning, disponibles avec sklearn.

Le projet est segmenté en 3 parties. Une partie d'extraction de features, une partie d'Entrainnement et en fin une partie d'Evaluation.

Voici quelques points que nous avons vu :

  • Gestion du dataset : Equilibrage des données avec Upsampling
  • Selection des Best Features, pour un meilleur apprentissage
  • Méthode d'apprentissage : Train - Test - Validation Set
  • Export du model : Pickle file

Durant ce projet, j'ai appris beaucoup sur le Machine Learning, de l'apprentissage d'un model à l'évaluation de données. J'ai vu des méthode pour traiter les features et le dataset. J'ai pu participer sur le projet de bout en bout, ce qui était très instructif.
